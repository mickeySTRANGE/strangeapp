  • 新刊通知くん
    • 設定したキーワードで楽天ブックス、楽天Koboを毎日検索します。新しい本がショップに追加されるとプッシュ通知、メール通知で教えてくれます。
     
  • 蟲神器デッキシミュレーター（非公式）
    • 大創出版様より発売されているトレーディングカードゲーム「蟲神器」のデッキシミュレーターです。
     
  • ステップタイマー
    • タイマーの時間の間に、いくつか途中経過をおしえてくれるタイマーです。通知方法は太鼓の音（メディア再生音量に依存）、バイブレーション、プッシュ通知（マナーモード、通知音量に依存）の3通りです。
     
  • ポケモンGoレイドDPS計算機
    • ポケモンGoのレイドバトルでのDPS計算機です。ボスポケモンを選択すると、ポケモンとわざの組み合わせをDPSの高い順に表示します。
